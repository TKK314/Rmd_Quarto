1.2 段組み

コード:

x <- c(1, 2, 3, 1, 2)
y <- c("A", "A", "A", "B", "B")

paste0(x, y)

結果:

[1] "1A" "2A" "3A" "1B" "2B"

上記のように記載することで、横並びにすることができる

  • 結果のみを右側に出力することも可能

    • 同じコードを記載して、eval: falseで実行しない、echo: falseでコードを表示しないというふうにできる

コード:

x <- c(1, 2, 3, 1, 2)
y <- c("A", "A", "A", "B", "B")

paste0(x, y)

結果:

[1] "1A" "2A" "3A" "1B" "2B"

1.3 Rのtable出力

以下のようにknitrを用いることでhtml形式でtableを出力でき、webページにした時に見た目が良くなる

library(datasets)
library(tidyverse)
iris %>% 
  head() %>% 
  knitr::kable(format = "html", digits = 5, align = "c",
               caption = "sample table")
sample table
Sepal.Length Sepal.Width Petal.Length Petal.Width Species
5.1 3.5 1.4 0.2 setosa
4.9 3.0 1.4 0.2 setosa
4.7 3.2 1.3 0.2 setosa
4.6 3.1 1.5 0.2 setosa
5.0 3.6 1.4 0.2 setosa
5.4 3.9 1.7 0.4 setosa

通常のようにdata.frameを出力すると以下のようになってしまう

iris %>% head()
  Sepal.Length Sepal.Width Petal.Length Petal.Width Species
